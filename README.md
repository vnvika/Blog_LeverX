# Blog
## Stages and time of work:
### 1. Registration 
   - email confirmation ~ `2h`
   - save code in redis ~ `3h`  
   - password recovery ~ `1h`
### 2. Authorization 
   - jwt token ~ `12h`
### 3. Interacting with the application ~ `12h`
### 4. Testing ~ `2h`

## How to run application?
write in the console:

`mvn clean package`

`mvn spring-boot:run`
